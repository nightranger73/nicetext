# nicetext
A simple script to output a message using a ttf font on terminal using imagemagick and chafa

## Usage:

```
$> nicetext [OPTIONS]

Options:
  -f FONT          Font name (example: 'DejaVu-Sans-Bold')
  -m MESSAGE       Text to output
  -c COLOR         Color scheme: sky (def), fire, forest, neon, sunset, ocean, cyberpunk
  -s SIZE          Font size in points (def: 72)
```

## Requirements
```
imagemagick
chafa
```

This scripts can be used as a replacement for banner text like figlet or toilet enhancing the look of your terminal, as a greeting at your login screen
